numpy.where(条件, [ x , y , ] / )
根据 condition 从 x 或 y 中选择返回的元素。

a = np.array([[0, 1, 2],
              [0, 2, 4],
              [0, 3, 6]])
np.where(a < 4, a(条件为真时返回原值), -1(否则返回该值))  # -1 is broadcast
>>>array([[ 0,  1,  2],
       [ 0,  2, -1],
       [ 0,  3, -1]])
